COURSE PURPOSE

In this course learners will learn how to design, build and deliver an Information Systems that will meet the needs of business. Learners will learn to utilize system development life cycle management techniques.

EXPECTED LEARNING OUTCOMES

Upon completion of the course, the learner should be able to:

  1. 1.Define various types of information systems concepts and terminologies.
  2. 2.Describe the different phases of Systems Development Life Cycle (SDLC).
  3. 3.Apply specific modeling methodology to analyze, design and specify requirements for a business situation.

  4. 4.Transform requirements specifications into practical and achievable design specifications.

    COURSE DESCRIPTION

    Introduction; IT System Development Cycle; Conception; Feasibility Study; Systems Investigation; Fact-finding Methods; Analysis of Investigation; System requirements; Data collection methods; Questionnaires; Interviews; Systems Design; Distributed vs. Centralized File systems; Security and Audit; System Study Report; CASE tools and analysis of using CASE tools.
